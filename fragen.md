Was trifft auf GIT zu

[ ] Jede „version stellt eine komplette Kopie dar
[ ] Benutzer müssen nicht von einem Zentralen Repository aus Arbeiten
[ ] Arbeiten ist nur mit Verbindung zum Lead Server möglich

Wie sind die einzelnen Commits miteinander verbunden  
[ ] Der Parent wert eines commit zeigt auf den SHA 1 value des vorherigen
[ ] Der Parent wert eines commit zeigt auf die commit ID des vorherigen
[ ] In der commit Table sind die commit Id´s in der richtigen reihenfolge eingegeben
[ ] Der Child value des letzten commit wird mit der commit ID des nachfolgenden commits überschrieben. 
 
Was ist der HEAD ? 
[ ] Ein Pointer der immer auf den aktuellsten commit im aktuellen Branch zeigt
[ ] Ein Pointer der immer auf den ersten zuletzt deployten commit im aktuellen Branch zeigt
[ ] Metainformationen am Anfang eines commit
[ ] Wichtige Metainformationen über das repository
